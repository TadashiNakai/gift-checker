# GIFT Checker

**GIFT 形式テキストの文法チェック・動作確認ツール**

🔗 [ツールを開く](https://tools.nakaix.com/gift-checker/) | [GitHub](https://github.com/TadashiNakai/gift-checker)

---

## 概要

Moodle の GIFT 形式で記述した問題ファイルを、ブラウザだけで文法チェック・プレビュー・動作確認できるシングルファイル HTML ツールです。外部サーバーへのデータ送信は一切行いません。

---

## 主な機能

### 文法チェック
- 選択肢ブロック内の行頭が `=` または `~` 以外になっている箇所を検出
- 選択肢テキスト内の未エスケープ制御文字（`=` `~` `{` `}`）を検出
- 1問に複数の `{…}` がある場合（標準 GIFT では2つ目以降は空欄にならない）を警告
- 問題箇所の行番号を表示し、クリックでテキストエリア内の該当行へジャンプ

### 問題一覧
- GIFT テキストをパースして問題一覧テーブルを表示
- 種類・タイトル・問題文冒頭・全体解説の有無を一覧表示
- 行クリックで任意の問題から解答を開始

### 対応する GIFT 問題タイプ（全9種）

| タイプ | 説明 |
|--------|------|
| 単一選択（MC） | `=` 正解・`~` 不正解、`%nn%` 重み付け対応 |
| 複数選択（Multiple answers） | チェックボックス、部分点対応 |
| 正誤（True/False） | ○/× ボタン、2種フィードバック対応 |
| 記述・短答（Short answer） | 別解・大小無視・重み付き部分点 |
| マッチング（Matching） | プルダウン選択、各組の正誤表示 |
| 穴埋め（Missing word） | 文中の単一空欄に対応 |
| 数値（Numerical） | 許容誤差・範囲（`low..high`）・複数回答 |
| 記述式（Essay） | 自動採点なし、解答例・解説を表示 |
| 説明（Description） | 回答なし、説明文として表示 |

その他、`$CATEGORY:`・`::タイトル::`・`####` 全体解説・`//` コメント行・`\=` 等のエスケープにも対応しています。

### 問題演習
- 問題ナビゲーションボタン（`[ 1 ] [ 2 ] … [ n ]`）でどの問題にも自由にジャンプ可能
- シャッフル時はボタンにパース時の元番号を表示（順番がわかる）

### 答え合わせ・解説
- 正解・不正解・部分正解の判定
- 選択した選択肢へのフィードバックと問題全体の解説を表示
- オプション「答え合わせ画面に問題文・選択肢も表示する」で上下一体表示

### 結果まとめ
- 全問終了後（または途中でも）スコア画面に遷移
- 全問題のタイトル・種類・正誤を一覧表示（未回答は「－」）
- 行クリックで問題の詳細確認（確認モード）へ移動可能

### 確認モード
- 問題文・選択肢（正誤色分け）・解説を表示
- 選択肢にマウスオーバーで個別フィードバックをツールチップ表示
- 結果まとめへ戻るボタンのみ（再解答不可）

### その他
- GIFT ファイルをテキストエリアにドラッグ＆ドロップして読み込み（バイナリ検出あり）
- 4つのトグル設定（問題順シャッフル・選択肢シャッフル・ヒント表示・問題文表示）をブラウザに記憶（localStorage）
- 完全ブラウザ完結・シングルファイル・外部通信なし

---

## 使い方

1. [ツールを開く](https://tools.nakaix.com/gift-checker/)（または `index.html` をブラウザで開く）
2. GIFT 形式のテキストをテキストエリアに貼り付けるか、ファイルをドラッグ＆ドロップ
3. **「📋 問題一覧を確認」** をクリック
   - 文法エラーがあれば該当箇所が赤字で表示される
4. 一覧画面で設定を選択し、**「▶ 最初から問題を開始」** または行クリックで演習開始
5. 全問終了後、**「📊 結果まとめを見る」** で正誤一覧を確認
