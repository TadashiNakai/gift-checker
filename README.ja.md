## GIFT Checker

[English](README.md) ｜ **日本語**

**GIFT 形式テキストの文法チェック・演習・HTML出力ツール（日本語／英語対応）**  

[![Version](https://img.shields.io/badge/version-0.29-blue)](https://tools.nakaix.com/gift-checker/)
[![License](https://img.shields.io/badge/license-MIT-green)](LICENSE)

🔗 [ツールを開く](https://tools.nakaix.com/gift-checker/) | [GitHub](https://github.com/TadashiNakai/gift-checker)

### 概要

Moodle の GIFT 形式で記述した問題ファイルを、ブラウザだけで文法チェック・プレビュー・演習シミュレーション・HTML出力できるシングルファイル HTML ツールです。外部サーバーへのデータ送信は一切行いません。日本語・英語の切り替えに対応しています。

GIFT を Moodle に取り込む**前**に、ブラウザ上で次のことができます。

- 文法エラーの検出（取り込み前の事前チェック）
- 全9問題タイプの**演習シミュレーション**（実際に解いて答え合わせ・フィードバック確認）
- 問題用紙＋解答キーを**1枚の HTML ファイルで書き出し**（画像・音声・動画の埋め込みにも対応）
- 結果まとめを HTML として保存
- 書式・画像・音声・動画を含む GIFT のプレビューと編集
- 必要に応じて、問題表示画面および HTML 出力で HTML タグを反映（raw HTML モード。危険なタグ・属性は除去）

### 主な機能

#### 文法チェック

- 選択肢ブロック内の行頭が `=` または `~` 以外になっている箇所を検出
- 選択肢テキスト内の未エスケープ制御文字（`= ~ { }`）を検出
- 選択肢のフィードバック（`#` 以降）内の未エスケープ `= ~` を検出（新しい選択肢と誤解釈される箇所）
- 同一の選択肢・別解が重複している箇所を検出
- 1問に複数の `{…}` がある場合（標準 GIFT では2つ目以降は空欄にならない）を警告
- 問題箇所の行番号を表示し、クリックでエディタ内の該当行へジャンプ

#### 問題一覧

- GIFT テキストをパースして問題一覧テーブルを表示
- 種類・タイトル・問題文冒頭・全体解説の有無を一覧表示
- 問題文冒頭は HTML タグやエンティティ化されたタグ（例：`&amp;lt;font...&amp;gt;`）を外し、プレーンテキストとして表示
- 行クリックで任意の問題から解答を開始
- 一覧画面で次の実行オプションを設定可能
  - 問題順シャッフル
  - 選択肢シャッフル
  - 選択肢ヒント表示
  - 答え合わせ画面に問題文・選択肢を表示
  - 問題表示と HTML 出力で HTML タグを反映（raw HTML モード。`&lt;...&gt;` / `&amp;lt;...&amp;gt;` 形式も復元）

#### 対応する GIFT 問題タイプ（全9種）

| タイプ | 説明 |
|---|---|
| 単一選択（MC） | `=` 正解・`~` 不正解、`%nn%` 重み付け対応 |
| 複数選択（Multiple answers） | チェックボックス、部分点対応 |
| 正誤（True/False） | ○/× ボタン、2種フィードバック対応 |
| 記述・短答（Short answer） | 別解・大小無視・重み付き部分点 |
| マッチング（Matching） | プルダウン選択、各組の正誤表示、ダミー選択肢（対応する小問のない余分な誤答）に対応 |
| 穴埋め（Missing word） | 文中の単一空欄に対応 |
| 数値（Numerical） | 許容誤差・範囲（`low..high`）・複数回答 |
| 記述式（Essay） | 自動採点なし、解答例・解説を表示 |
| 説明（Description） | 回答なし、説明文として表示 |

その他、`$CATEGORY:`・`::タイトル::`・`#### 全体解説`・`// コメント行`・`\=` 等のエスケープ・`\n`（改行）エスケープにも対応しています。また、Moodle からの書き出しで各解答（`=` / `~` の直後）やそのフィードバックに付与される `[moodle]` / `[html]` 等のフォーマット指定子は、表示・採点の対象ではないメタ情報として自動的に取り除きます。

#### 演習シミュレーション

- 問題ナビゲーションボタン（`[ 1 ] [ 2 ] … [ n ]`）でどの問題にも自由にジャンプ可能
- 番号ボタンの手前に移動ボタン `[ |◀ ] [ ◀ ] [ ▶ ] [ ▶| ]`（最初へ・前へ・次へ・最後へ）を配置し、問題間を素早く移動できる（先頭・末尾では対応するボタンが自動的に無効化される）
- シャッフル時はボタンにパース時の元番号を表示（順番がわかる）
- 正解・不正解・部分正解の判定
- 選択した選択肢へのフィードバックと問題全体の解説を表示
- オプション「答え合わせ画面に問題文・選択肢も表示する」で上下一体表示
- 答え合わせ画面の「← 問題文に戻る」で同じ問題の問題文画面（選択・入力済みの状態）に戻って選び直し可能
- 問題画面の「前へ」ボタンは、一つ前の問題が解答済みなら「← 前の解説に戻る」（その問題の最新の採点結果へ）、未解答なら「← 前の問題に戻る」とラベルが切り替わり、`◀`（単に前の問題文へ移動）との役割の違いが分かる
- 選び直して答え合わせをすると、その問題の得点だけが最新の選択内容で上書きされ、合計点には常に各問題の最後の解答が反映される

#### 結果まとめ・確認モード

- 全問終了後（または途中でも）スコア画面に遷移し、全問題の正誤を一覧表示（未回答は「－」）
- 記述式（Essay）・説明（Description）は自動採点の対象外として扱われ、正答率の分母（配点）から除外される
- 問題ナビゲーションボタンは解答状況に応じて下線色が変化し、正解は緑・不正解は赤・採点対象外（記述式・説明）は灰色で一目で把握できる
- 解き直して結果が変われば色も連動して更新される
- 行クリックで問題の詳細確認（確認モード）へ移動
- 確認モードでは問題文・選択肢（正誤色分け）・解説を表示
- 選択肢のマウスオーバーで個別フィードバックをツールチップ表示
- **「⬇ 結果をHTMLで保存」** ボタンで、得点率・一覧表に加えて「不正解だった問題」「自動採点されなかった問題（記述式・説明）」の詳細を単一 HTML ファイル（`result-YYYYMMDDhhmmss.html`）として書き出し

#### 印刷・配布用 HTML 出力

- **「⬇ 問題と解答をHTMLで保存」** ボタンで、問題用紙＋解答キーを単一 HTML ファイル（`quiz-YYYYMMDDhhmmss.html`）として書き出し
- 前半に問題、後半に解答をまとめて配置（印刷を想定したコンパクトなレイアウト）
- 選択肢は丸数字（①〜㊿）、マッチングは左を丸囲み英字（Ⓐ Ⓑ …）・右候補を丸数字で表示し、解答は `Ⓐ→③` … の形式で対応を明示
- 一覧画面のシャッフル設定（問題順・選択肢）を反映
- 画像は data URL として埋め込まれ、1ファイルで完結（追加ファイル不要）
- 保存 HTML 内の画像は縦横比を維持し、本文幅を超える場合のみ縮小
- 音声・動画を含む場合も、対応する media 要素として HTML に埋め込み
- 一覧画面の「HTMLタグをそのまま反映する」オプションがオンの場合、問題表示画面と HTML 出力に HTML タグを反映（script、on* 属性、javascript: URL などは除去）

#### リッチテキスト入力

- 入力欄は `contenteditable` のリッチテキストボックス
- ブラウザウィンドウが大きい場合は入力欄も広く・高く表示
- 最小サイズを保ちつつ、縦方向の手動リサイズにも対応
- ツールバーから太字（B）・斜体（I）・下線（U）・上付き・下付きを適用可能
- Word 等からの貼り付けで書式を保持、選択してツールバーボタンで手動適用も可能
- 書式付きの問題には `[html]` マーカーが自動付与されてエクスポートされる
- プレビュー画面にも書式が反映される
- エディタ内の画像サイズは、問題表示画面での表示サイズに近くなるよう、元の `style` / `width` / `height` を尊重

#### 画像・音声・動画の扱い

##### 画像

- クリップボードからの画像貼り付けに対応（PNG・JPEG 等）
- 画像データは内部でメモリ管理し、GIFT テキスト上にはプレースホルダーとして扱う設計
- 画像を含む問題には `[html]` マーカーが自動付与される
- プレビュー・演習・HTML 出力すべてに画像が表示される
- GIFT エクスポート時は `<img src="data:image/...">` 形式で埋め込まれる
- Moodle GIFT エクスポートに見られる `src\=` / `data\:` 形式のエスケープにも対応
- 保存 HTML では画像の縦横比を維持し、過度な高さ制限を行わない

##### 音声

- `<audio src="data:audio/mp3;base64,..."></audio>` 形式に対応
- `<audio><source src="data:audio/mp3;base64,..."></audio>` 形式に対応
- Moodle GIFT 由来の `src\=` / `data\:` 形式のエスケープにも対応
- 問題表示画面・HTML 出力で再生コントロール付き音声として表示

##### 動画

- `<video src="data:video/mp4;base64,..."></video>` 形式に対応
- `<video><source src="data:video/mp4;base64,..."></video>` 形式に対応
- `video/mp4` の base64 埋め込みを HTML 出力でも保持
- 問題表示画面・HTML 出力で再生コントロール付き動画として表示

#### HTML タグの反映（raw HTML モード）

一覧画面の **「HTMLタグをそのまま反映する」** オプションをオンにすると、問題文や選択肢内に含まれる HTML 風の記述をブラウザに渡して描画します。  
Moodle XML や GIFT 由来で `&lt;font&gt;`、さらに二重エンコードされた `&amp;lt;font&amp;gt;` のような形式になっている場合も、表示時に復元してから処理します。

例：`&lt;font color="#FF0000"&gt;①②&lt;/font&gt;`  
この設定をオンにすると、問題表示画面と HTML 出力では ①② が赤色で表示されます。

raw HTML モードでは、`<font>` の入れ子、`<span style="...">`、`<sup>` / `<sub>`、`<b>` / `<i>` / `<u>` などの書式タグも、タグ単位で分解せずブラウザの HTML パーサーに任せて描画します。  
一方で、安全性のため、`script`、`style`、`onclick` などの `on*` イベント属性、`javascript:` URL は除去または無効化します。

問題一覧の「問題文（冒頭）」だけは、見やすさを優先し、raw HTML モードのオン／オフに関係なく HTML タグを外したプレーンテキストとして表示します。

タグの閉じ忘れがある場合はブラウザが補正しますが、意図しない範囲まで書式がかかる可能性があります。必要な GIFT ファイルでのみオンにすることを推奨します。

#### GIFT 形式でのエクスポート

- **「⬇ GIFT 形式をダウンロード」** ボタンで `quiz-YYYYMMDDhhmmss.gift` ファイルをダウンロード
- 書式タグ・`<font>`・画像・音声・動画を含むブロックには `[html]` が自動付与される

#### 日本語／英語の切り替え

- ヘッダーの **「日本語 / English」** トグルで UI 言語を切り替え
- 初回はブラウザの言語設定で自動判定（日本語環境は日本語、それ以外は英語）
- 手動で選んだ言語は localStorage に記憶され次回以降優先される
- サンプル問題・HTML 出力も選択言語に追従
- 問題一覧・演習・結果まとめ・確認モードなど、表示中の画面も切り替えと同時に再描画され、ラベルや採点結果の表示が現在の言語に即時反映される

#### その他

- GIFT ファイルをエディタにドラッグ＆ドロップして読み込み（バイナリ検出あり）
- 書式・画像・音声・動画を含む既存 GIFT ファイルを読み込むと、エディタ上で可能な範囲で復元
- トグル設定（問題順シャッフル・選択肢シャッフル・ヒント表示・問題文表示・安全なHTMLタグ有効化）をブラウザに記憶（localStorage）
- 完全ブラウザ完結・シングルファイル・外部通信なし

### 使い方

1. [ツールを開く](https://tools.nakaix.com/gift-checker/)（または `index.html` をブラウザで開く）
2. GIFT 形式のテキストを入力欄に貼り付けるか、ファイルをドラッグ＆ドロップ
3. **「📋 問題一覧を確認」** をクリック
4. 文法エラーがあれば該当箇所が赤字で表示される
5. 一覧画面で設定を選択し、目的に応じて操作する
6. **「▶ 最初から問題を開始」** または行クリックで演習シミュレーション
7. **「⬇ 問題と解答をHTMLで保存」** で問題用紙＋解答キー HTML を書き出し
8. **「⬇ GIFT 形式をダウンロード」** で GIFT ファイルを保存
9. 演習後は **「📊 結果まとめを見る」** で正誤一覧を確認し、必要に応じて **「⬇ 結果をHTMLで保存」** で結果を書き出す

### 他ツールとの違い

GIFT を扱うツールの多くは「問題の生成・他形式への変換」が中心です。GIFT Checker は、Moodle に取り込む**前のテキスト段階**で、(1) 全9問題タイプの演習シミュレーション、(2) 問題用紙＋解答キーの HTML 出力、(3) 画像・音声・動画を含む教材のブラウザ内確認を1ファイルのブラウザツールで提供する点に特徴があります。いずれも完全にローカルで動作し、外部送信はありません。
